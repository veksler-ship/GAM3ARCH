# GAM3ARCH — модель игрового выгорания

> **4 зоны. 1 симуляция. 1 ритуал — и выгорание уходит.**

Фреймворк для анализа поведения игроков и предотвращения выгорания.  
На основе **512 игроков**, **Python-симуляции** и **личного опыта в WoW (Nerub’ar Palace, 2025)**.

## Зоны игрового цикла
Horizon (пауза)
↑
Back (выход) ← Nexus (общение)
↑
Forge (гринд)

- **Forge** — рейды, фарм, "ещё один пул"  
- **Nexus** — Discord, стримы, чаты  
- **Back** — ритуал: "бай, спасибо"  
- **Horizon** — сон, жизнь, перезапуск  

> **Impotence** — когда ты в Forge, но кайфа нет.
