# GAM3ARCH

Framework for analyzing player burnout and behavioral zones in games (Forge, Nexus, Back, Horizon).

## Overview
GAM3ARCH — простая модель, которая помогает анализировать, как игроки выгорают и восстанавливаются в разных фазах игрового цикла.

## Zones
- **Forge** — активная игра, гринд, напряжение.
- **Nexus** — общение, стримы, взаимодействие.
- **Back** — завершение, выход, ритуалы.
- **Horizon** — пауза, отдых, возвращение.

## How to run
```bash
pip install -r requirements.txt
python run.py
```
